# AboutRipan

This repository contains the source code for **AboutRipan**, my personal website and portfolio. The site serves as a central place to share my background, projects, and technical work, and to present my profile as a software engineer.

Live site: https://www.aboutripan.com

---

## Overview

AboutRipan is a personal portfolio website designed to clearly communicate who I am, what I build, and how I approach problem-solving. The site focuses on simplicity, accessibility, and performance while highlighting key projects and experiences.

---

## Features

- Clean, minimal, and responsive design
- Dedicated sections for About, Projects, and Contact
- Project highlights with concise technical summaries
- Optimized for readability and accessibility across devices
- Lightweight and fast-loading

---

## Technologies Used

- React
- Tailwind
